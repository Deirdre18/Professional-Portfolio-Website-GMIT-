Professional Portfolio Project

This is my professional portfolio containing to demonstrate skills developed through the BSc Web Technologies and Programming Course at GMIT, Galway. My journey on this course was not always smooth and I came across many difficult areas to navigate, which gave me an incredible impetus to keep learning and growing.

This project involves the migration of projects covered during first and second semester. In some cases I have selected just a handful of projects and in other cases have listed all projects/assignments/etc for a particular module, that I particularly enjoyed learning.

This project seeks to build a professional portfolio and will be shared on version control (Git Hub) with academic staff and any other collaborators, such as employers, etc. In order to navigate through this project, click on the links, which will then bring you to a page containing information about a particular module (assessment, etc). The course was continuous assessment thoughout the entire year and every module was assessed in accordance with continuous assessment criteria. Where a module was not completely passed at the end of each semester, there was an opportunity to repeat the failed assessment and to pass the modules. It is a requirement to get the degree that ALL modules are successfully passed.
